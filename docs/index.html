About the Project

Atomic Launcher is an open-source initiative by Dariusz Mirończuk, focused on delivering a compact, extensible launcher tailored for small indie games and educational projects. Built with Python, the repository includes everything needed: tools for asset importing, dynamic game manifests, audio management, and modern UI components.
